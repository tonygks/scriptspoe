[Flags]

#######################
Дефолтный фильтр
#######################
Новые фильтры автоматически не добавляются(подгрузка через код). Возможно редактирование порога оценивания, веса и т.п. в существующих фильтрах


;Boots minvalue maxvalue craftvalue prefixes/suffixes weight target						
;						
!ClassType	Boots					
!Score	350					 - количество очков для подсветки
!ScoreHits4	250					- количество очков при 4 аффиксах/статах из списка ниже для подсветки
!ScoreHits5	200					- тоже самое для 5 аффиксов/статов
А дальше статы, минимальные максимальные значения, крафт  и т.д. через табуляцию(поддается настройке через одно место)
